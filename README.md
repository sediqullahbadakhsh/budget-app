## Getting Started

### Clone this repository

```bash
$ git clone https://github.com/sediqullahbadakhsh/recipe-app-group-project.git
$ cd recipe-app-group-project
```

### Install gems

```bash
$ bundle install
```

### Setup database

```bash
$ rails db:create
```

### Start server

```bash
$ rails s
```

Open http://localhost:3000/ in your browser

### Run tests

```bash
$ bundle exec rspec
```
